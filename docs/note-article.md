# 【無料ツール】モール併売でカテゴリ変換に困っている方へ。LLMで自動マッピングするWebアプリを公開しました

## はじめに

メルカリShops、楽天市場、ヤフーショッピングなど、複数のECモールで商品を販売している方なら、誰もが一度は経験したことがあるのではないでしょうか。

**「この商品、どのモールのどのカテゴリに登録すればいいんだろう...」**

モールごとにカテゴリ体系が異なるため、1つの商品を複数のモールに登録する際、毎回カテゴリを調べて手動で設定するのは本当に時間がかかります。特に商品数が多い場合、この作業だけで1日が終わってしまうことも。

そんな課題を解決するため、**LLM（大規模言語モデル）を活用したカテゴリ自動マッピングツール**を開発・公開しました。今回は、このツールの特徴や使い方をご紹介します。

## モール併売におけるカテゴリ変換の課題

### なぜカテゴリ変換が大変なのか

複数のECモールで商品を販売する際、以下のような課題に直面します：

1. **カテゴリ体系の違い**
   - メルカリShops、楽天市場、ヤフーショッピングでは、それぞれ独自のカテゴリ体系を持っています
   - 同じ商品でも、モールごとに適切なカテゴリが異なることがあります

2. **カテゴリIDの管理**
   - 各モールには固有のカテゴリIDが存在します
   - 商品登録時には、正確なカテゴリIDを指定する必要があります

3. **時間とコスト**
   - 1商品あたり3つのモールに登録する場合、カテゴリを調べるだけで数分かかります
   - 100商品ある場合、カテゴリ設定だけで数時間かかってしまいます

4. **人的ミスのリスク**
   - 手動でカテゴリを設定するため、間違ったカテゴリを選択してしまうリスクがあります
   - カテゴリミスは、商品の検索順位や売上に直接影響します

## LLMを活用した自動マッピングツールの特徴

### 最新のAI技術「RAG」を採用

このツールは、**Google Gemini API**を活用し、**RAG（Retrieval-Augmented Generation）**という最新のAI技術を採用しています。

RAGとは、関連する情報を検索してからAIに回答させる技術です。単純にAIに質問するのではなく、まず各モールのカテゴリ一覧から関連するカテゴリを検索し、その中から最も適切なカテゴリを選択します。これにより、**高精度なマッピング**が実現できます。

### 主な機能

1. **商品名からの自動マッピング**
   - 商品名を入力するだけで、各モールの適切なカテゴリを自動的に割り当てます
   - 最大30件まで一括処理可能（31件以上の場合はお問い合わせフォームからご相談ください）

2. **3つのモールに対応**
   - メルカリShops
   - 楽天市場
   - ヤフーショッピング

3. **カテゴリIDとフルパスを自動取得**
   - 商品登録に必要なカテゴリIDを自動的に取得
   - カテゴリの階層構造を表すフルパスも表示

4. **エクスポート機能**
   - 結果をクリップボードにコピー
   - CSV形式でダウンロード可能

### なぜLLMを使うのか

従来のルールベースのマッピングツールでは、以下のような限界がありました：

- 商品名のバリエーションに対応できない
- 新しい商品カテゴリに対応できない
- モールごとのカテゴリ体系の違いを理解できない

一方、LLMを活用することで：

- **文脈を理解したマッピング**: 商品名の意味を理解し、適切なカテゴリを選択
- **柔軟な対応**: 新しい商品やカテゴリにも対応可能
- **高精度な結果**: RAGにより、関連カテゴリのみをコンテキストに含めることで精度向上

## 使い方

### ステップ1: 商品名を入力

改行区切りで商品名を入力します。最大30件まで一括処理できます。

```
NIKE ロゴ刺繍キャップスナップバック
フルーツオブザルーム 90s ビンテージフォトプリントTシャツ
無印良品 コットンストレッチパンツ
```

### ステップ2: マッピング実行

「マッピング実行」ボタンをクリックすると、各モールのカテゴリが自動的に割り当てられます。

### ステップ3: 結果を確認・エクスポート

マッピング結果を確認し、必要に応じてクリップボードにコピーまたはCSVダウンロードします。

## 技術的な特徴

### RAGによる精度向上

このツールでは、以下のような流れでマッピングを行います：

1. **商品名からキーワードを抽出**
2. **関連カテゴリを検索**（最大50件）
3. **LLMが関連カテゴリの中から最適なものを選択**

これにより、単純にAIに質問するよりも、**はるかに高精度なマッピング**が実現できます。

### 並列処理による高速化

各モールへのマッピングを並列で実行することで、処理速度を大幅に向上させています。

### 軽量モデルの採用

商品名からのマッピングには、**Gemini 2.0 Flash Lite**という軽量モデルを使用しています。これにより、高速かつ低コストでマッピングを実行できます。
